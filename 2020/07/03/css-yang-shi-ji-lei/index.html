css样式积累

  • 图标的样式设置一般用span标签
    /*
        背景图标一般用backgroud,no-repeat为不平铺
        background-size: contain；使其宽高完全填充至内容区域
    */
    background: url('/imgs/icon-cart-checked.png') no-repeat center;
    background-size: contain;
  • 鼠标移上去时切换图片，header-logo相当于相框，a标签相当于相册，存放两张图片切换（before和after）
    .header-logo{
                        display: inline-block;
                        width: 55px;
                        height: 55px;
                        background-color: #FF6600;
                        a{
                            display: inline-block;
                            width: 110px;
                            height: 55px;
                            &:before{
                                content: ' ';   //给图片留白,一定要留白占位置
                                display: inline-block;
                                width: 55px;
                                height: 55px;
                                background: url('/imgs/mi-logo.png') no-repeat center;
                                background-size: contain;
                                transition:margin .2s;
                            }
                            &:after{
                                content: ' ';
                                display: inline-block;
                                width: 55px;
                                height: 55px;
                                background: url('/imgs/mi-home.png') no-repeat center;
                                background-size: contain;
                            }
                            //鼠标移上去的时候切换图片，前面的图片向左移动55px
                            &:hover:before{
                                margin-left: -55px;
                                transition:margin .2s;  //过渡动画效果
                            }
                        }
                    }
  • cursor: pointer; //鼠标放上去就会出现一个小手（a标签自带这样的效果）
  • transition: all .5s; //所有元素出现一个0.5s的动画效果

scss

  • scss的引入使css样式更加灵活，把需要重复使用的css样式放在scss文件中供我们调用
  • 通过import的方式导入 ： @import “./../assets/scss/base.scss”;
  • 既可以直接调用，又可以通过函数封装进行调用
    //定义
    @mixin flex($fd:row,$hov:space-between,$col:center){
        display: flex;
        flex-direction: $fd;
        align-items: $col;
        justify-content: $hov;
    }

    //调用，可以传参，不传参就用默认值
    @include flex();

隐藏和显示

  • 如何制造一个鼠标移上去就显示，不移上去就隐藏的效果,先设置高度为0，透明度为0，鼠标移上去的时候显示高度和透明度
    &:hover {
                color: $colorA;
                .children {
                  height: 220px;
                  opacity: 1;
                }
              }
              .children {
                position: absolute;
                top: 112px;
                left: 0px;
                width: 1226px;
                border-top: 1px solid #e5e5e5;
                box-shadow: 0px 7px 6px 0px rgba(0, 0, 0, 0.11);
                z-index: 10;
                height: 0;
                opacity: 0;         //透明度为0
                overflow: hidden;   //内容会被修剪，并且其余内容是不可见的。
                transition: all .5s;
                .product {
                  float: left;
                  height: 220px;
                  width: 16.6%;
                  font-size: 12px;
                  line-height: 12px;
                  text-align: center;
                  position: relative;
                  a {
                    display: inline-block;
                  }
                  img {
                    width: auto;
                    height: 111px;
                    margin-top: 26px;
                  }
                  .product-img {
                    height: 137px;
                  }
                  .product-name {
                    font-weight: bold; //字体加粗
                    margin-top: 19px;
                    margin-bottom: 8px;
                    color: $colorB;
                  }
                  .product-price {
                    color: $colorA;
                  }
                  &:before{
                      content: ' ';
                      position: absolute;
                      top: 28px;
                      right: 0px;
                      border-right: 1px solid $colorF;
                      height: 100px;
                      width: 1px;
                  }
                  &:last-child::before{
                      display: none;    //最后一个before元素不显示
                  }
                }
              }
  • 还可以用block和none来显示和隐藏 block为显示 none为不显示
    //本来显示为隐藏，鼠标移上去的时候显示
    display: none;

    &:hover {
                background-color: $colorA;
                .children {
                  display: block ;
                }
              }

如何作一个弹框动画css样式难点

  • 如何是弹框一直居中，注意position和transform要搭配使用
    position: fixed;    //fixed固定定位，相对于浏览器
    top: 40%;
    left: 50%;
    width: 660px;
    height: auto;

    transform: translate(-50%,-50%);    //往上（x轴）,左（y轴）移动自身长宽的 50%，以使其居于中心位置
  • absolute和fixed的区别
    • fixed:固定定位
    • absolute:绝对定位
    • 1、没有滚动条的情况下没有差异
    • 2、在有滚动条的情况下，fixed定位不会随滚动条移动而移动，而absolute则会随滚动条移动
  • 放大效果 transform: scale(1.5); //放大1.5倍

视频播放时需要注意的css样式

  • object-fit: cover; //保持原有尺寸比例。宽度和高度中短的那条边跟容器大小一致，长的那条等比缩放。可能会有部分区域不可见。
  • outline: none; //元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用,这里的作用是去掉这条线

如何实现吸顶功能

  • 首先我们要知道当前滚动条的高度，当滚动条的高度高于要实现吸顶功能的模块时吸顶，不然不变
    methods: {
        initHeight() {
          //获取当前滚动条所处的高度
          let scrollTop =
            window.pageXOffset ||
            document.documentElement.scrollTop ||
            document.body.scrollTop;
          this.isFixed = scrollTop > 151 ? true : false;
        }
      },
  • 监听滚动条的事件
    mounted() {
        //一个监听滚动条的事件
        window.addEventListener("scroll", this.initHeight);
      },
  • 当退出这个页面时摧毁方法
    destroyed() {
        window.removeEventListener("scroll", this.initHeight, false);
      }
  • 设置一个样式，滚动条滑动到模块以下时显示
    :class="{'is-fixed':isFixed}"


    &.is-fixed {
    position: fixed;
    top: 0;
    width: 100%;
    box-shadow: 0px 5px 5px $colorE; //阴影x轴横向为0，y轴竖向为5px，宽度为5px
    }

添加浮动通用浮动样式

  • 页面虽然可以使用flex弹性布局，但是有时候还是要用到float浮动样式，我们可以先设置定义好全局通用，要用的时候就拿来用就是了
  • 记住使用浮动最后要记得清除浮动
  • clearfix 是在父组件上设置
    .fl {
        float: left;
    }

    .fr {
        float: right;
    }

    .clearfix:before,
    .clearfix:after {
        content: ' ';
        display: table;
    }

    .clearfix:after {
        clear: both;
    }
  • 使用
    <div class="item-version clearfix">
        <h2>选择版本</h2>
        <div class="phone fl" :class="{'checked':version==1}" @click="version=1">6GB+64GB 全网通</div>
        <div class="phone fr" :class="{'checked':version==2}" @click="version=2">4GB+64GB 移动4G</div>
      </div>

点击选择切换框

  • 在挑选货物的时候会有很多个样式供我们选则，我们可以点击切换，选择不同的版本，version默认为1，:class=”{‘checked’:version==1}”的意思就是version为1的时候才会显示这个样式
    <div class="item-version clearfix">
        <h2>选择版本</h2>
        <div class="phone fl" :class="{'checked':version==1}" @click="version=1">6GB+64GB 全网通</div>
        <div class="phone fr" :class="{'checked':version==2}" @click="version=2">4GB+64GB 移动4G</div>
      </div>
